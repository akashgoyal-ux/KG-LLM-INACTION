# ch17_fin — Financial Investigative Copilot

## Objective
Analyst-facing Streamlit app combining KG investigation, Graph RAG,
ownership maps, event timelines, and what-if exposure probes.

## Panels
- Entity profile (LegalEntity / Instrument)
- Ownership map (network visualization)
- Event timeline
- Exposure path explorer
- Q&A panel (Graph RAG with citations)

## Safety
- Cypher generated by LLM is parsed + validated against allow-list
- Read-only Neo4j role for the app
